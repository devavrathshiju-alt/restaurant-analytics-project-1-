# Restaurant Analytics Project

## 📌 Project Overview
This project analyzes restaurant data to understand ratings, costs, cuisines,
and service availability using SQL and Power BI.

## 🛠 Tools Used
- Python (Data Cleaning)
- RunSQL (SQL Analysis)
- Power BI (Dashboard)
- GitHub (Version Control)

## 📊 Key Insights
- Majority of restaurants are Average Rated
- High-rated restaurants are concentrated in specific areas
- Online ordering is more common than table booking
- Higher-rated restaurants generally have higher average costs

## 📁 Project Structure
- data/: cleaned dataset
- sql/: SQL queries
- powerbi/: Power BI dashboard
- screenshots/: dashboard image
